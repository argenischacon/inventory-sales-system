# Sistema de Inventario y Ventas

API REST completa para la gestión de inventario y ventas.

## Descripción

Este proyecto es una aplicación Spring Boot diseñada para gestionar inventario y ventas de manera eficiente.

## Tecnologías Utilizadas

*   Java 21
*   Spring Boot 3.3.3
*   PostgreSQL / H2 (Base de datos)
*   MapStruct
*   JWT (JSON Web Token)
*   OpenAPI / Swagger UI
